UXON: User eXperience Ontology Network based system

UXON is an ontology-based system to support user experience assessment in the context of an immersive experience application for large events. It aims to automate the user experience evaluation process, with data collected through indirect observation and the use of metrics, and to make the measurement results available in the form of tables and graphs, allowing the evaluator to analyze them and generate evaluation reports on immersive experiences.
